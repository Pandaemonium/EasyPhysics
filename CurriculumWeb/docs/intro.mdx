# Start Here

EasyPhysics is a concept-first map of fundamental physics for curious learners who want meaning before jargon.

The project starts from a simple spine:

```text
state -> change -> symmetry -> conservation -> field -> particle -> interaction -> geometry
```

The goal is not to make physics shallow. The goal is to make the structure visible. Names arrive after the idea has a foothold, and math appears as a tool for saying the idea precisely.

## How to Read

Begin with the MVP path if you want the first guided sequence. Each lesson is short, ends with a small knowledge check, and may offer an optional Math Lens. You can skip the math lens and continue without losing the main thread.

Each article should help you answer three things:

- What is the core idea?
- What problem does it solve?
- How does it connect to the rest of physics?
