# Math Lens: Functions as Rules

A <Term name="function">function</Term> is an input-output rule. You give it something. It gives back something determined by the rule.

```text
input -> rule -> output
```

In algebra, you might see:

```text
f(x) = x + 2
```

That means: give the rule a number, and it returns a number 2 larger.

For physics, the idea becomes more interesting when the input is a state.

```text
state now -> rule of change -> state later
```

The "state now" can contain several values: position, velocity, temperature, field value, or something more abstract. The rule can also be simple or complicated.

## What the Symbols Are Doing

A <Term name="variable">variable</Term> is a changeable slot. In `f(x) = x + 2`, the symbol `x` is a slot for whatever input you choose.

The power of functions is that one rule can handle many cases. You do not need a separate instruction for every possible input.

That is one reason equations are so compact. A small line of math can describe a whole family of changes.
