# Change: The Rule That Moves a State

A state is a snapshot. Physics also wants the movie rule.

If the state says where a ball is and how it is moving right now, the <Term name="change-rule">rule of change</Term> says what states are allowed to come next. In standard language, this is part of <Term name="change-rule">dynamics</Term>.

The difference matters:

The rule does not have to be complicated to be powerful. "Keep moving at the same velocity unless something changes that motion" is already a rule. So is "temperature spreads from warmer regions toward cooler neighboring regions." So is "a quantum state changes according to its energy rule."

The state tells you where the story is. The change rule tells you how the story can continue.

## Why This Is a Big Deal

Once we separate state from change, we can ask better questions.

- What information belongs in the state?
- What rule changes that state?
- What parts of the rule stay the same when we shift our viewpoint?
- What quantities survive the motion?

Those last two questions are where symmetry enters.
