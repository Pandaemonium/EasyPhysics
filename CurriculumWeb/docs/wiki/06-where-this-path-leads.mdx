# Where This Path Leads

You now have the first small engine:

<ConceptFlow />

A <Term name="state">state</Term> says what the model needs now. A change rule says how that state can move. A symmetry is a change that leaves the rulebook unchanged. A conservation law is a quantity protected by that sameness.

This is not the whole of physics. It is the beginning of a map.

## The Same Engine Reappears

A <Term name="field">field</Term> is what happens when the state is spread across space: a value here, another value nearby, and rules for how those values change together.

A <Term name="particle">particle</Term>, in the modern picture, is not a tiny bead added on top of fields. It is a stable, countable ripple of a field.

An <Term name="interaction">interaction</Term> is a rule for how systems or fields affect one another.

<Term name="geometry">Geometry</Term> becomes physics when the rules of distance, time, and path are part of what the universe is doing. That is the door into relativity.

Quantum mechanics will make the word "state" deeper. Relativity will make "same for different observers" deeper. Field theory will make "particle" deeper.

But the path remains recognizable.
